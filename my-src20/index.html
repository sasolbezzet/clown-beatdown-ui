Clown Beatdown – my‑src20

This repository contains a sample SRC20 token and the Clown Beatdown contract, designed to run on the Seismic Testnet.

How to use

  1. Copy .env.example to .env and fill in the values: 
    RPC_URL=https://gcp-1.seismictest.net/rpc   # Seismic Testnet RPC endpoint
PRIVATE_KEY=0xYOUR_PRIVATE_KEY                # Account that will deploy/interact
  2. Install dependencies: 
    npm ci
  3. Deploy the contracts (replace YOUR_CONTRACT_ADDRESS after deployment): 
    npm run interact   # runs src/interact.js which deploys & interacts
  4. After deployment you will have two contract addresses:
    • Clown Beatdown – address you will interact with (hit, add secret, rob, etc.)
    • SRC20 – the token used for rewards
  5. Use the src/interact.js script or any EVM‑compatible wallet (MetaMask, viem, ethers) to call the contract functions.
    Example (with viem): 
    import { createPublicClient, http } from 'viem'
import { sepolia } from 'viem/chains'

const client = createPublicClient({
  chain: sepolia,
  transport: http(process.env.RPC_URL),
})
// then call contract methods as shown in src/interact.js

Repository layout

  • src/ClownBeatdown.sol – the game contract.
  • src/SRC20.sol – a minimal ERC‑20 token used for rewards.
  • src/interact.js – helper script to deploy and interact with the contracts using ethers/viem.
  • .env.example – template for environment variables.

For the full Seismic Testnet documentation see https://docs.seismic-testnet.io.
